# PhraseLauncher

定型文を素早く入力できる定型文ランチャー

Windowsでは、Windowsキー + Vでピン留めやクリップボードの履歴の入力ができますが  
1. 登録できる数が少ない  
2. 定型文特化でない  
3. ショートカットによる素早い入力ができない(マウスや、スクロールして選んで入力)  
4. グループ分けできない  
などの問題があったため自作しました。

## 使い方（クイックスタート）
0. 最新バージョンをダウンロードします
   https://github.com/takuyash/PhraseLauncher  
   
1. exe を実行するとタスクトレイに常駐します  

2. タスクトレイのアイコンを右クリックして、設定画面が開けます

   グループの登録や定型文の登録ができます  
   
3. Ctrl 2回連続 もしくは Ctrl + Shift + O のショートカットでランチャーを呼び出すことができます  

4. 表示された項目にキーが割り当てられているためそのキーを入力することでマウスやカーソル移動することなく入力できます

## 起動方法
・exeを実行すると、タスクトレイに常駐します。  
・タスクトレイのアイコンを右クリックして、設定画面が開けます。  
・ Ctrl 2回連続 もしくは Ctrl + Shift +  O で表示できます。  

## 特徴
1. 定型文に0–9 + A–Z のキー割り当てで超高速入力
表示された項目には自動で 0〜9、A〜Z のキーが割り当てられます。  
そのキーを押すことで高速に入力できます。  

3. 不要な設定がない
不要な設定機能はないシンプルなアプリです。  

4. 定型文編集登録機能
Shift + Enter で改行できます。
